# MKDev Portfolio

**MKDev Portfolio** is a modern, responsive portfolio template designed to showcase web development projects and case studies. It features clean layouts, customizable sections, and live project links, making it ideal for developers to present their work professionally. Built using **HTML, CSS, and JavaScript**, this portfolio is open-source and easy to deploy.

---

## Key Features

- ✅ Fully responsive and mobile-friendly  
- ✅ Customizable project and case-study pages  
- ✅ Live project previews with GitHub code links  
- ✅ Modern design with developer-friendly aesthetics
