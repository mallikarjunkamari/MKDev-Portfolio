# Certification Images Setup

To add your certification images:

## Required Images:
Place the following images in this `assets/certifications/` folder:

1. **java-se-programmer.png** - Your Oracle Java SE Programmer certification image
2. **aws-certified-developer.png** - Your AWS Certified Developer certification image
3. **spring-framework.png** - Your Spring Framework certification image
4. **python-certification.png** - Your Python Developer certification image (already added)

## Image Specifications:
- **Format**: PNG (preferred) or JPG
- **Size**: Recommended 400x300 pixels or similar aspect ratio
- **Quality**: High resolution, clear text
- **Background**: White or transparent preferred

## Sources for Certification Images:
- Download from certification provider's website
- Take screenshots of your digital certificates
- Use official certification badges/logos

## Alternative:
If you don't have certification images, you can:
- Use certification logos
- Create custom certificate designs
- Use placeholder images until you obtain the real ones

The certification cards will automatically display these images once you add them to this folder.
